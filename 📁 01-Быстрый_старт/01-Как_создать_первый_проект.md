# 🚀 Как создать первый проект в Bytrox

> **Добро пожаловать в экосистему!** Этот гайд поможет вам инициализировать рабочую среду, получить ключи доступа и сделать первый тестовый запрос к нашему API менее чем за 5 минут.

---

## 1. Инициализация в консоли

Для начала работы вам необходимо зарегистрировать ваше приложение в единой панели управления Bytrox. Это создаст изолированный контейнер для ваших данных.

1. Перейдите в **[Bytrox Console](https://app.bytrox.com)** и авторизуйтесь.
2. В левом навигационном меню выберите раздел **«Проекты»**.
3. Нажмите кнопку **Создать проект** в правом верхнем углу.
4. Введите название (например, `Bytrox Bot Backend`) и выберите регион серверов.

---

## 2. Настройка среды и безопасности

Каждый проект в Bytrox использует архитектуру Zero-Trust. Выберите правильный уровень доступа для ваших эндпоинтов в зависимости от среды разработки:

| Среда выполнения | Уровень WAF | Доступ к Bytrox Drive | Описание |
| :--- | :--- | :--- | :--- |
| **Development** | Низкий | Только чтение | Локальная разработка и тестирование интеграций. |
| **Staging** | Средний | Чтение / Запись | Предрелизная среда. IP-адреса должны быть в Whitelist. |
| **Production** | Строгий | Полный доступ | Боевая среда. Максимальная защита от DDoS и фильтрация пакетов. |

---

## 3. Генерация ключей и первый запрос

После создания проекта система выдаст вам `Project_ID` и секретный `API_Token`. 

`Критически важно:` Никогда не коммитьте ваш токен в публичные репозитории. Храните его исключительно в переменных окружения `.env`.

### Пример интеграции через Python (FastAPI)

Чтобы проверить работоспособность, вы можете отправить базовый запрос к платформе. Пример реализации контроллера для проверки статуса:

```python
from fastapi import FastAPI, Header, HTTPException
import httpx

app = FastAPI(title="Bytrox Client")

BYTROX_API_URL = "[https://api.bytrox.com/v1](https://api.bytrox.com/v1)"
SECRET_TOKEN = "Ваш_Секретный_Токен_Из_Консоли"

@app.get("/system/status")
async def check_bytrox_status():
    async with httpx.AsyncClient() as client:
        response = await client.get(
            f"{BYTROX_API_URL}/ping",
            headers={"Authorization": f"Bearer {SECRET_TOKEN}"}
        )
        
        if response.status_code != 200:
            raise HTTPException(status_code=response.status_code, detail="Ошибка API")
            
        return {"status": "success", "bytrox_response": response.json()}
